## 📌 项目简介
本系列实验涵盖数字电子技术的两个核心内容：
- **门电路逻辑测试**：74LS00（与非门）、74LS20（四输入与非门）、74LS86（异或门）的逻辑功能验证与波形分析
- **FPGA应用设计**：基于74161计数器的任意进制计数器设计，包含分频器、译码器和七段数码管显示

**亮点**：静态测试 + 动态波形分析 + FPGA完整信号链设计

## 💻 实验二：FPGA任意进制计数器设计

### 2.1 系统架构

```
时钟信号(CLK) → 分频器 → 计数器(74161) → 译码器(74LS47) → 七段数码管
```

### 2.2 设计原理

#### 计数器 74161 功能
- 4位同步二进制计数器
- 具有同步并行预置、异步清零和计数使能功能
- 时钟上升沿触发
- 计数值达1111时进位输出(RCO)有效

#### 分频器作用
- 降低输入时钟频率，适配数码管显示
- 通过计数器实现2ⁿ分频

#### 译码器 74LS47
- 低有效输出，驱动共阳极数码管
- 将4位BCD码转换成7段数码管段码

### 2.5 关键技术要点

| 模块 | 功能 | 实现方式 |
|------|------|----------|
| 分频器 | 时钟降频 | 计数器计数时钟边沿，达到阈值输出脉冲 |
| 计数器 | 任意进制计数 | 74161同步预置/异步清零实现模N计数 |
| 译码器 | BCD转7段码 | 74LS47输出低有效，驱动共阳极数码管 |
| 显示 | 数字显示 | 七段数码管动态/静态显示 |

---

### 2.6 思考题：大容量计数器的设计方法

**同步级联法**：
- 所有计数器共用同一时钟
- 前一级RCO作为下一级ENT/ENP使能
- ✅ 优点：计数速度快，无级联延迟
- ❌ 缺点：需要较多逻辑控制

**异步级联法**：
- 前一级RCO作为下一级时钟输入
- ✅ 优点：电路简单
- ❌ 缺点：存在级联延迟，适用于低速应用
